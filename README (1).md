# Gerador de Senhas Seguras

## 1. Introdução
Este documento apresenta o desenvolvimento de um Gerador de Senhas Seguras como parte da atividade avaliativa da disciplina Engenharia de Software II. O objetivo consiste em implementar uma ferramenta capaz de gerar senhas personalizáveis e seguras, considerando boas práticas de segurança da informação.

## 2. Objetivos do Projeto
O projeto visa:
- Proporcionar ao usuário a geração de senhas fortes;
- Permitir personalização da senha por meio da seleção de tipos de caracteres;
- Avaliar a força da senha gerada;
- Demonstrar na prática o uso de HTML, CSS e JavaScript.

## 3. Estrutura do Sistema
A aplicação é composta pelos seguintes arquivos:

```
Gerador de Senhas/
│── img/
│── index.html
│── styles.css
│── script.js

```

## 4. Funcionalidades Implementadas
### 4.1 Personalização
O usuário pode selecionar:
- Quantidade de caracteres;
- Inclusão de letras maiúsculas e minúsculas;
- Inclusão de números;
- Inclusão de símbolos.

### 4.2 Avaliação da Força da Senha
O sistema avalia a senha em níveis:
- Fraca
- Média
- Forte

### 4.3 Usabilidade
- Interface intuitiva;
- Componentes interativos.

## 5. Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## 6. Como Executar
1. Faça o download do projeto.
2. Abra o arquivo **index.html** em qualquer navegador.
3. Utilize os controles disponíveis para gerar a senha desejada.

## 7. Possíveis Melhorias
